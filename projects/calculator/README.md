# Advanced Console Calculator

## Описание
Этот калькулятор умеет:
- Вычислять арифметические выражения с приоритетом операций, скобками и возведением в степень (`^`).
- Использовать константы `pi` и `e`.
- Решать линейные уравнения вида `a*x + b = c`.
- Решать квадратные уравнения вида `a*x^2 + b*x + c = 0` (выводит вещественные или комплексные корни).
- Работать с комплексными числами при делении на ноль (использует модуль `cmath`).

## Примеры использования
Введите выражение: (2+3)*4^2Результат: 80.0
Решить линейное уравнение? (да/нет): даВведите уравнение: 2*x + 5 = 13x = 4.0
Решить квадратное уравнение? (да/нет): даВведите уравнение: x^2 + 1 = 0Корни: -0.0 + 1.0j, -0.0 - 1.0j
