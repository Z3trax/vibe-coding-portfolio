# Express Server Project

Простой веб-сервер на Express.js для демонстрации статических файлов и API.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run start
```

Для разработки с автоматической перезагрузкой:

```bash
npm run dev
```

## API

- `GET /api/projects` — возвращает список проектов
- `GET /api/projects/:id` — возвращает один проект по id

## Статические файлы

Открыть в браузере:

- `http://localhost:3000`
