# V1be_T3st_Bot

Простой Telegram-бот на Python с использованием aiogram 3.x.

## Функционал

- Отвечает на команду `/start` приветствием.
- На любое текстовое сообщение отвечает эхом (повторяет текст).
- Использует асинхронный подход.
- Включает логирование и обработку ошибок.

## Установка

1. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

2. Запустите бота:
   ```
   python main.py
   ```

## Структура проекта

- `main.py` - основной код бота
- `requirements.txt` - зависимости
