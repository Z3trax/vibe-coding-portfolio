# 🕐 Интерактивный Таймер Обратного Отсчёта

Современный таймер обратного отсчёта с поддержкой светлой и тёмной темы, созданный на чистом JavaScript, HTML и CSS.

## 🎯 Возможности

- ✅ **Ввод времени** — два отдельных поля для минут (0-59) и секунд (0-59)
- ✅ **Управление** — кнопки Старт, Пауза и Сброс
- ✅ **Отображение** — формат MM:SS в крупном монопространственном шрифте
- ✅ **Обратный отсчёт** — точный отсчёт вниз с интервалом 1 секунда
- ✅ **Завершение** — сообщение "🎉 Время вышло!" и звуковой сигнал
- ✅ **Валидация** — защита от отрицательных чисел и некорректного ввода
- ✅ **Защита** — при изменении времени во время работы таймер автоматически сбрасывается
- ✅ **Темы** — переключение между светлой и тёмной темой с сохранением в localStorage
- ✅ **Адаптивность** — учтён дизайн для мобильных устройств

## 🚀 Использование

1. Откройте `index.html` в браузере
2. Введите минуты и секунды в поля ввода
3. Нажмите **Старт** для начала обратного отсчёта
4. Используйте **Пауза** для остановки (время сохранится)
5. Нажмите **Сброс** для возврата к 00:00

- **Современный интерфейс** с использованием градиентов и теней

## 💻 Технические детали

- **Без зависимостей** — использован только нативный JavaScript
- **Web Audio API** — для звукового оповещения при завершении
- **localStorage** — для сохранения предпочтения тёмной темы
- **CSS переменные** — для удобного переключения тем

## 🔧 Функции JavaScript

### `CountdownTimer`
Главный класс, управляющий логикой таймера:

- `start()` — запуск таймера
- `pause()` — пауза таймера
- `reset()` — сброс в 00:00
- `updateDisplay()` — обновление отображения времени
- `playAlertSound()` — воспроизведение звукового сигнала
- `toggleTheme()` — переключение светлой/тёмной темы
- `validateInput()` — валидация пользовательского ввода

## 🎵 Звуковой сигнал

При завершении таймера воспроизводится звук (частота 800 Hz, тип sine, продолжительность 0.5 сек).
Если браузер не поддерживает Web Audio API, сигнал будет проигнорирован (без ошибок).

## ⌨️ Требования к вводу

- **Минуты**: 0-59
- **Секунды**: 0-59
- Отрицательные числа автоматически преобразуются в 0
- Числа больше максимума обрезаются
- Нечисловой ввод обрабатывается корректно

## 🌙 Сохранение предпочтения темы

Выбор темы (светлая/тёмная) сохраняется в браузер и восстанавливается при следующем посещении.
