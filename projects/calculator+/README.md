# Advanced Calculator GUI

Графическая версия калькулятора с поддержкой:
- Арифметических выражений со скобками и степенью (`^`)
- Констант π и e
- Решения линейных уравнений вида `a*x + b = c`
- Решения квадратных уравнений вида `a*x² + b*x + c = 0` (вывод вещественных и комплексных корней)
- Комплексных чисел (при делении на ноль)

## Интерфейс
- **Стандартный режим**: ввод выражения с помощью кнопок или клавиатуры, результат выводится снизу.
- **Линейное уравнение**: поля для ввода коэффициентов a, b, c, кнопка «Решить».
- **Квадратное уравнение**: аналогично.
