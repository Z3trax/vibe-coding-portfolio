# vibe-coding-portfolio

Привет! Я начинающий разработчик, осваиваю подход **vibe coding** — разработку с активным использованием AI-инструментов (DeepSeek, Copilot, Codex). В этом репозитории собраны мои учебные проекты, которые демонстрируют навыки работы с Python, JavaScript, созданием веб-сайтов, телеграм-ботов, API и игр.

## Цель
Научиться быстро и качественно создавать работающие прототипы и полноценные приложения для фриланс-заказов. Каждый проект сопровождается промптами, которые я использовал для генерации кода, а также моими комментариями.
